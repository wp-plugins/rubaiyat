== Description ==

A Wordpress plugin which displays a random verse from the <a href="http://en.wikipedia.org/wiki/Rubaiyat_of_Omar_Khayyam" >Rubaiyat of Omar Khayyam</a> on admin pages. The plugin incorporates text from Edward FitzGerald&#8217;s translation (source: <a href="http://www.gutenberg.org/etext/246" >Gutenberg project</a>) and is based on the Hello Dolly plugin distributed with Wordpress.
== Installation ==

1. Upload `rubaiyat.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
